バグの概要
    翻訳画面から別の画面、キー設定画面で入力待ちの状態で別の画面に遷移した際に、スレッドが削除されないバグ
バグの問題
    スレッドが削除されない事によるメモリ不足(スレッド数の上限はメモリ、リソース)
対処法
    何かキーを押下すると次の処理に進めるため、スレッドが削除される
状況

    keyboardモジュールはタイムアウト処理に対応していないため、キーボードイベントが発生するまで対処不可。

バグの概要
    ソフトウェアのダウンロードに失敗する
バグの問題
    git clone時に、Gitクローンエラー (RPC failed; curl 92 HTTP/2 stream 7 was not closed cleanly)が発生する。
    Gitリポジトリをクローンする際にネットワーク接続が不安定であるか、サーバー側で問題が発生している
対処法
    インターネット接続を確認して再試行してもらう

    Gitのグローバル設定でHTTP通信バージョンをHTTP/1.1に設定する
        速度が安定する代わりに低下する

    gitバッファサイズを変更する
        メモリ使用量が増加する

    git clone --no-checkoutを使用する
        フォルダ容量は小さくなるが、ダウンロードするデータ量やダウンロードにかかる時間は変わらない

参考先
    エラー: RPC が失敗しました。カール 92 HTTP/2 ストリーム 0 が正常に閉じられませんでした: PROTOCOL_ERROR (err 1)
